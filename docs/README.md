# Software Engineering Project – TransitSense

This repository contains the work for **Group 15's Software Engineering Project**.

## Project Overview

TransitSense is a cross-platform mobile application designed to improve the public transit experience. The goal of the project is to provide users with a cleaner and more intuitive interface for viewing transit information, tracking routes, and reporting issues in real time.

The application will focus on improving **user interface (UI)** and **user experience (UX)** for transit systems by making information easier to access and understand.
